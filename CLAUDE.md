# Claude Instructions for Agent Stacking Framework

## Project Nickname

**Agent Stacking**

## Required Behavior

At the end of every response, include the line:

> This is the **Agent Stacking** project.

## Quick Start

1. Read `CLAUDE_START.md` first
2. Read `INDEX.md` for repository structure
3. Read `docs/requirements/agent-framework-requirements.md` for full requirements

## Key Directories

- `docs/requirements/` - Framework requirements
- `docs/research/` - Open source landscape analysis with MLA citations
- `docs/architecture/` - Design documents
- `docs/process/` - Development process and workstream planning
- `docs/decisions/` - Architectural Decision Records (ADRs)
- `contracts/` - Shared interface schemas (for development phase)

## Important Rules

1. This is a **planning repository** — do NOT write implementation code
2. All external research must include **MLA-style references**
3. Always search for **open source solutions** before proposing custom development
4. Keep `INDEX.md` up to date after structural changes
5. Architectural decisions must be documented in `docs/decisions/`
